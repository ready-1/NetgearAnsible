# Artifacts Table of Contents

## Purpose of this directory
This directory is meant to be a central location of Netgear M4300 Series switch documentation and prompt/code examples for reference.  The CLI documentation has been cleaned and is assumed 90% authoritative.

## Artifacts Listing
1. command_ref.jsonl - command reference containing all available commands in JSONL format 
2. M4300_CLI_EN_chunks.jsonl - the full text of the CLI manual converted from pdf presented in ~2000 token chunks with overlaps.
3. M4300_CLI_EN.txt - the full text of the M4300 CLI manual pdf presented as text extraction
4. M4300_Switch_CLI_Exercise.md - Summary of Learnings from M4300 Switch CLI Exercise
