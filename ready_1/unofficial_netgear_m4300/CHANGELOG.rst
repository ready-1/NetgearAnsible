Unofficial Netgear M4300 Collection

v1.0.0
======

Release Summary
---------------

Initial release of the Unofficial Netgear M4300 Ansible collection.

New Features
------------

- Telnet connection plugin for Netgear M4300 switches
- System configuration module supporting:
  - Management IP configuration
  - SSH service setup and key generation
  - Local user management
  - SNTP server configuration
  - SNMP settings
- Comprehensive CLI command reference documentation
- Module utilities for common Netgear switch operations

Breaking Changes
----------------

None

Bugfixes
--------

None

Known Issues
------------

- SSH and REST API connection plugins not yet implemented
- VLAN, interface, and routing modules not yet implemented
